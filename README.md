# 📦 물류 센터 작업 관리 시스템 (CLI)

이 프로그램은 물류 센터의 작업자가 할당받은 업무를 관리하고 수행할 수 있도록 돕는 CLI 애플리케이션입니다.

## 🚀 주요 기능 및 메뉴 설명

사용자는 프로그램을 실행한 후 다음과 같은 메뉴를 통해 작업을 진행할 수 있습니다.

### 1. 공통 메뉴 (Common)

* **`0. 도망치기`**
    * 현재 진행 중인 모든 작업을 중단하고 즉시 프로그램을 종료합니다.
    * 완수하지 못한 작업이 남아 있어도 조건 없이 종료가 가능합니다.

* **`1. 작업자 정보 보기`**
    * 현재 로그인된 작업자의 상세 상태를 확인합니다.
    * **표시 항목:** 작업자명, 총 할당받은 업무량, 현재까지 완수한 업무량, 그리고 실시간 **작업 진행률(%)**을 계산하여 보여줍니다.
    * 자신의 현재 페이스를 확인하고 퇴근까지 얼마나 남았는지 가늠할 수 있습니다.

* **`2. 작업 더 할당하기`**
    * 기존 할당량 외에 추가 업무를 부여합니다.
    * 추가할 수량을 입력하면 전체 할당 업무량이 증가합니다.

* **`3. 퇴근 하기`**
    * 할당받은 모든 업무를 완수했을 때만 사용 가능합니다.
    * 업무를 다 마치지 않았다면 퇴근할 수 없다는 메시지가 출력됩니다.

### 2. IB (Inbound) 전용 메뉴

* **`11. 포장 박스 호수 추천받기`**
    * 상품명(`phoneCase`, `cup`, `snack`)을 입력하면 해당 상품에 가장 적합한 규격의 박스 호수를 추천해줍니다.
* **`12. 포장하기`**
    * 실제로 상품을 포장하는 단계입니다. 상품명과 박스 호수를 입력하며, 추천받은 호수와 일치해야만 '완료'로 처리됩니다.

---

## 🛠 실행 시나리오 예시

1. **정보 입력:** 이름과 오늘 목표 작업량을 입력합니다.
2. **메뉴 선택:** 상황에 맞는 메뉴 번호를 입력합니다.
3. **작업 수행:** 포장 등의 업무를 진행하며 완료량을 채웁니다.
4. **상태 확인:** '작업자 정보 보기'를 통해 진행도를 체크합니다.
5. **종료:** 모든 작업을 마치고 '퇴근 하기'를 누르거나, 힘들면 '도망치기'를 누릅니다.
